# Mobile-First Financial Net Worth Tracker

A sleek, responsive, mobile-first web application designed to track and visualize your personal financial net worth over time. Built with Python, Flask, SQLite, and Tailwind CSS.

## Features

- **Mobile-First Dashboard:** A vertically stacked, responsive dashboard optimized for smartphone screens, displaying your Total Net Worth, Total Assets, Total Liabilities, and Quarter-over-Quarter Growth.
- **Interactive Charting:** Visualizes your historical net worth trend using a beautifully styled Chart.js line graph.
- **Detailed Breakdowns:** Tap on "Total Assets" or "Total Liabilities" to view a complete breakdown of individual accounts and their most recent balances.
- **Historical Filtering:** Tap on any individual account to drill down into its specific transaction history.
- **CRUD Functionality:** Easily add new records, or edit and delete historical entries directly from your mobile device.
- **Long-Format Data Model:** Ensures clean, scalable data storage with a normalized SQLite schema.

## Technology Stack

- **Backend:** Python 3, Flask, Flask-SQLAlchemy (SQLite)
- **Frontend:** HTML5, Jinja2, Tailwind CSS (via CDN)
- **Charting:** Chart.js

## Local Setup & Installation

### Prerequisites
- Python 3.8+ installed on your system.

### Steps
1. **Clone the repository:**
   ```bash
   git clone https://github.com/PaulAnderie/NetWorthTracker.git
   cd NetWorthTracker
   ```

2. **Create a virtual environment:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```

3. **Install the dependencies:**
   ```bash
   pip install Flask Flask-SQLAlchemy python-dateutil
   ```

4. **Initialize the Database:**
   This script will drop existing tables and populate the database with 2 years of simulated quarterly data for demonstration purposes.
   ```bash
   python init_db.py
   ```

5. **Run the Application:**
   ```bash
   python app.py
   ```

6. **Access the App:**
   Open your browser and navigate to `http://127.0.0.1:5000/`. For the best experience, open your browser's developer tools and switch to a mobile device view.
